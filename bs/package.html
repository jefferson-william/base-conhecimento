Comments are not allowed in the package.json file (See http://blog.getify.com/json-comments/ for why this is silly), so they are added here. The package.json file is responsible for installing npm packages. See the official package.json specification for more information.
  • name - This is used if you want to create your own npm package. Unfortunately, this is required by NPM. It does not matter what goes here too much.
  • version - Same as above.
  • devDependencies - The npm packages your project depends on. Gulp packages start with 'gulp'. These are downloaded to a node_modules folder at the root of the project.
